# ドキュメント概要

## ドキュメント構造

### 使用者向け文書

- **`../README.md`** - プロジェクトの概要、ビルド方法、使用方法

### 開発履歴・現状

- **`DEVELOPMENT.md`** - 開発計画とロードマップ

### 実装計画

- **`next_update.md`** - 次期アップデート計画

### 設計文書

- **`coding_rules.md`** - 開発ルールと原則
- **`concept.md`** - プロジェクトの基本コンセプト

## 文書管理方針

### 更新タイミング

- 実装完了時: `DEVELOPMENT.md` に記録
- 新機能計画時: `next_update.md` を更新

### アーカイブ

- 完了した計画文書は `docs/archive/` に移動
- バージョン別の履歴として保存

### 除外項目

実装対象外として明確に除外。
